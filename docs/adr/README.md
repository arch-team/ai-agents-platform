## ADR 模板

新建 ADR 时复制以下模板：

```markdown
# ADR-NNN: 标题

- **日期**: YYYY-MM-DD
- **状态**: 提议 | 已采纳 | 已废弃 | 已替代
- **关联**: ADR-XXX (如有)

## 背景

什么情况下需要做这个决策？

## 决策

选择了什么方案？

## 理由

为什么选择这个方案？考虑了哪些备选方案？

## 影响

这个决策对项目的影响是什么？
```
