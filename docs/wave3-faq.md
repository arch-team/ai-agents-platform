# AI Agents 平台 FAQ — Wave 3 全公司推广版

> 本文档收录平台使用中的高频问题与解答，适用于所有员工。
> 平台地址：**生产环境** `http://ai-agents-prod-1419512933.us-east-1.elb.amazonaws.com`

## 一、基础使用

### Q: 什么是 AI Agent？和直接用 Claude 有什么区别？

Agent 是经过专项配置的 AI 助手，针对特定业务场景预设了指令、角色和行为边界。直接用 Claude 是通用对话，而 Agent 更像一个"岗位专员"——比如"代码审查 Agent"只关注代码质量，回答更专业、更聚焦。

### Q: 怎么开始一次 Agent 对话？

登录平台后，在首页选择一个 Agent 或模板，点击"开始对话"即可。第一次使用建议先浏览模板库，选择最接近需求的模板，比自己从零配置更快。

### Q: 对话有字数限制吗？单次提问可以多长？

单次消息建议不超过 2000 字，过长的输入可能导致响应时间增加或部分内容被截断。如需分析长文档，建议分段提交或使用支持文件上传的 Agent。

### Q: 对话记录会保存多久？在哪里查看历史对话？

对话记录默认保留 90 天。在平台左侧导航栏点击"历史对话"，可按时间或 Agent 名称筛选查看。超过 90 天的记录会自动归档，如需延长请联系管理员。

### Q: Agent 有没有记忆？上次对话的内容这次还记得吗？

默认情况下每次新建对话都是独立的，Agent 不会记住上次的对话内容。如果需要延续上下文，请在同一个对话会话中继续提问，或在新对话开头手动粘贴关键背景信息。

## 二、模板和 Agent 管理

### Q: 模板和自己创建 Agent 有什么区别？

模板是平台预置或团队共享的配置方案，可一键使用，适合大多数通用场景。自建 Agent 允许你自定义系统提示词、模型选择、温度参数等，适合有特定业务需求的高级用户。建议先用模板，不满足需求时再自定义。

### Q: 为什么有些模板我看不到？

模板的可见性分三种：**公开**（全员可见）、**团队可见**（仅所在团队）、**私有**（仅创建者）。如果看不到某个同事分享的模板，可能是权限设置为团队或私有，联系模板创建者将可见性改为"公开"即可。

### Q: 模型选哪个？Haiku 和 Sonnet 有什么区别？

| 模型 | 适用场景 | 响应速度 |
|------|---------|---------|
| Claude Haiku | 简单问答、格式转换、快速分类 | 快（约 1-3 秒） |
| Claude Sonnet | 复杂分析、代码生成、长文写作 | 中（约 3-8 秒） |

日常轻量任务推荐 Haiku，需要深度推理或高质量输出时选 Sonnet。

### Q: 如何修改已创建的 Agent 配置？

进入"我的 Agent"列表，点击对应 Agent 右侧的"编辑"按钮，可修改名称、系统提示词、模型和参数。修改后点击保存，新配置在下次对话时生效，不影响历史对话记录。

### Q: 可以把自己的 Agent 分享给团队吗？

可以。在 Agent 详情页找到"分享设置"，将可见性从"私有"改为"团队可见"或"公开"，并选择是否允许他人复制配置。分享后同事可以直接使用，但不能修改你的原始配置。

## 三、Agent Teams 功能

### Q: 什么是 Agent Teams？

Agent Teams 是 Wave 3 新上线的多 Agent 协同功能。你可以在一个对话中同时调用多个 Agent，让它们分工合作完成同一个任务。例如让"代码审查 Agent"负责发现问题，同时让"文档撰写 Agent"生成修复说明。

### Q: Agent Teams 适合什么场景？

典型场景包括：
- **代码 + 文档**：代码审查 Agent 找 Bug，文档 Agent 同步更新注释
- **数据 + 报告**：数据分析 Agent 提取洞察，报告撰写 Agent 生成 PPT 大纲
- **翻译 + 润色**：翻译 Agent 完成初稿，写作 Agent 进行母语级润色

适合需要多个专业视角交叉验证或流水线式处理的复杂任务。

### Q: 怎么创建一个 Agent Team 对话？

在平台首页点击"新建 Team 对话"，从已有 Agent 中选择 2-5 个成员加入 Team，设置协作模式（顺序执行或并行讨论），输入任务后点击发送。各 Agent 会分别给出输出，平台自动汇总展示。

### Q: Agent Team 中各 Agent 的输出如何区分？

每个 Agent 的回复前会标注其名称和角色标签，例如 `[代码审查 Agent]` 和 `[文档撰写 Agent]`。你可以在对话界面右侧的"成员面板"展开查看各 Agent 的独立输出，也可以查看平台生成的汇总视图。

### Q: Agent Teams 会更慢吗？

会有轻微延迟。顺序执行模式下响应时间为各 Agent 时间之和；并行模式下取最慢 Agent 的时间。建议 Team 成员不超过 3 个，且每个 Agent 使用 Haiku 模型以保证响应速度。

## 四、权限和安全

### Q: 我的数据安全吗？输入的内容会存到哪里？

所有对话数据存储在公司 AWS 账户内（us-east-1 区域），不会传输到 Anthropic 或第三方服务器用于模型训练。数据静态加密（AES-256），传输加密（TLS 1.2+）。对话内容仅你本人和平台管理员可查看。

### Q: 同事能看到我的 Agent 对话内容吗？

不能。对话记录默认私有，仅对话发起人可见。即使 Agent 是公开模板，其他用户的使用记录彼此隔离。管理员仅在安全审计或故障排查时有权限查阅，且会留下操作日志。

### Q: 谁可以看到我创建的 Agent？

Agent 的可见性由你控制：私有（默认）、团队可见、全员公开。进入 Agent 设置页随时可调整。删除 Agent 后对应的历史对话记录保留 30 天后清除。

### Q: 平台会用我输入的内容训练模型吗？

不会。本平台基于 Amazon Bedrock AgentCore 部署，采用 API 调用方式，输入内容不用于 Anthropic 或 AWS 的模型训练。具体数据处理协议参见公司 IT 安全政策。

## 五、常见问题排查

### Q: Agent 回答很慢，正常吗？

正常响应时间：Haiku 约 1-5 秒，Sonnet 约 5-15 秒。如果超过 30 秒无响应，可能是网络波动或服务负载高，建议刷新页面重试。持续缓慢请联系平台支持并附上对话 ID（页面右上角可复制）。

### Q: Agent 输出的内容需要我校验吗？

需要。AI 输出存在幻觉风险，可能生成听起来合理但实际有误的内容，尤其是涉及具体数字、引用来源、专业规范时。建议将 Agent 输出作为初稿，关键结论务必人工核实。

### Q: 为什么 Agent 没有理解我的意图，回答跑偏了？

尝试以下方法：① 在提问开头加背景说明；② 拆分复杂问题为多个步骤；③ 明确指定输出格式（如"用列表回答"）；④ 如果多次跑偏，考虑换用更匹配的模板或调整 Agent 的系统提示词。

### Q: 我想重新开始一个新对话，怎么操作？

点击对话界面左上角的"新建对话"按钮，或在侧边栏选择"清空当前对话"。注意清空后本次对话内容不可恢复，重要输出请提前复制保存。

### Q: 找不到某个功能在哪里？

平台功能入口：
- **模板库**：首页 → 浏览模板
- **我的 Agent**：左侧导航 → 我的 Agent
- **Team 对话**：首页 → 新建 Team 对话
- **历史记录**：左侧导航 → 历史对话
- **账号设置**：右上角头像 → 设置

## 六、账号和权限

### Q: 平台有哪些角色？各自有什么权限？

| 角色 | 权限说明 |
|------|---------|
| 普通用户 | 使用公开模板、创建私有 Agent、进行对话 |
| 团队管理员 | 额外可管理团队模板、查看团队使用统计 |
| 平台管理员 | 全平台管理、用户权限配置、系统监控 |

默认注册后为普通用户，团队管理员权限需申请。

### Q: 怎么申请更高权限或特殊功能访问？

发送邮件至平台支持邮箱（见本文末尾），注明：① 姓名和工号；② 申请的权限类型；③ 使用场景说明。团队管理员权限需直属上级审批，通常 1-2 个工作日内处理。

### Q: 离职员工的 Agent 和数据怎么处理？

HR 完成离职手续后，账号将在 24 小时内停用，对话数据保留 30 天后按数据保留策略清除。如需提前转移团队共享 Agent 的所有权，请在账号停用前联系团队管理员操作。
