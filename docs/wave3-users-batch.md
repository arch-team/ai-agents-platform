## 执行步骤（管理员操作）

### 方式一：API 批量创建（推荐）

使用管理员账号通过 Admin API 批量创建：

```bash
# 示例：批量创建单个用户（循环执行 20 次）
curl -X POST https://<platform-host>/api/v1/admin/users \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "tech_user_06",
    "email": "tech06@company.com",
    "password": "Wave3@2026!",
    "role": "DEVELOPER"
  }'
```

### 方式二：平台管理界面

1. 登录平台，进入 Admin Dashboard
2. 选择「用户管理」→「批量邀请」
3. 按部门逐批上传用户信息（CSV 格式：username,email,role）
4. 确认邀请邮件已发送

### 注意事项

- 创建前确认邮件服务器配置正常（用户会收到欢迎邮件）
- 初始密码 `Wave3@2026!` 需在首次登录后强制修改
- VIEWER 用户在独立创建第一个 Agent 后，可联系管理员申请升级为 DEVELOPER
- 建议先创建 5 人测试，确认邮件正常发出后再批量创建剩余用户
